## 📋 Project Overview

This Power BI dashboard was developed to monitor and analyze the **2025 Operational Plan** for the **Tabuk Branch** of the Saudi Red Crescent Authority (الهلال الأحمر السعودي).

It provides real-time insights into the achievement of strategic and operational objectives by comparing **Planned vs Actual** performance across quarters and months.

---

## 🎯 Strategic Objectives

- Achieving an Ideal Patient Journey
- Developing Organizational Structure
- Creating a Motivating Culture
- Community Integration & Asset Sustainability

---

## ✨ Key Features

- **Planned vs Actual** Comparison (Monthly & Quarterly)
- Performance by **Strategic Goals**, **Department Goals**, and **KPIs**
- Owner-wise Performance Analysis
- Variance Analysis (Time & Percentage)
- Interactive Slicers & Filters
- Progress Tracking with Traffic Light Indicators
- Executive Summary Dashboard

---

## 📊 KPIs & Metrics

- Number of Strategic Objectives: **4**
- Number of Department Objectives: **9**
- Number of Performance Indicators: **13**
- Number of Objective Owners: **3**

**Main Performance Indicators include:**
- Average Ambulance Response Time for Traffic Accidents
- Percentage of Emergency Calls Responded Within 8 Minutes
- CPR Success Rate (ROSC)
- Downtime due to Vehicle/Equipment Failure
- Employee Engagement Rate
- And more...
